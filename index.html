HTML Standard チートシート

プリアンブル

  • ドキュメントタイプの宣言をするタグ

    使用例

文書要素

  • HTML文書で有ることを宣言するタグ

    使用例

    使用例

  • 文書のメタ情報を指定するタグ

    使用例

    <meta ...>

  • スタイルシートを記述するタグ

    使用例

    <style>...</style>

フォーム

  • 入力フォームを作るタグ

    使用例

    <form>
    <input type="text"></input>
    <input type="submit></input>    
</form>

  • フォーム部品と項目名を関連付けるタグ

    使用例

    <form>
    <label for="age">年齢:</label>
    <input type="text" id="age">
    <input type="submit" value="送信">    
</form>

  • フォームの構成部品を作成するタグ

    使用例

    <form>
    <input type="text" name="name">
    <input type="password" name="pass" >
    <input type="submit" value="送信">    
</form>

  • ボタンを作成するタグ
